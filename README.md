# eventstats

FGO イベント報告の集計補助ツール

## 概要

- データ取得元: [FGO Harvest](https://fgojunks.max747.org/harvest/)
- 入力情報: Harvest のイベント報告ページ ID の一覧
    - 実行時に入力情報が記されたテキストファイルを指定する
    - Format: 各行は以下の形式: `<推奨レベル>\t<消費AP>\t<page-id>`
- 入力情報: イベントアイテムの定義
    - 実行時に入力情報が記されたテキストファイルを指定する

動作概要:

`https://fgojunks.max747.org/harvest/contents/quest/<page-id>.json` からイベント報告データを取得する。

## 注意事項

### イベントアイテムは枠数報告のみを集計対象とする

- 有効: みかん(x3): 50
- 無効: みかん: 150

### ポイント、QP は実際の獲得値に換算して集計する

ポイント(+500): 2, ポイント(+600): 4 => ポイント: 3400

## 管理画面概要

### 機能

- イベント管理
    - イベントの追加、編集、削除
    - イベントアイテムの登録、削除
- データ管理
    - 報告データの除外指定

### データ

- イベントが持つ情報
    - 期間
    - クエスト一覧
        - ID
        - クエスト名
        - 推奨レベル
        - 消費 AP
